## 🔤 BRAND FONTS

### Primary Font (Headings & Buttons)

**Font Name:** Inter
- **Weight:** Bold (700)
- **Use for:** Headings, buttons, important text
- **Alternatives if Inter not available:** Poppins Bold, Montserrat Bold

**In Canva:**
1. Search for "Inter" in font dropdown
2. Select "Inter Bold" or "Inter ExtraBold"
3. Set as "Heading font"

### Secondary Font (Body Text)

**Font Name:** Inter
- **Weight:** Regular (400)
- **Use for:** Body text, descriptions, paragraphs
- **Alternatives:** Poppins Regular, Open Sans

**In Canva:**
1. Search for "Inter" in font dropdown
2. Select "Inter Regular"
3. Set as "Body font"

### Font Sizes (For Reference)

**Headings:**
- H1: 48px (Bold)
- H2: 36px (Bold)
- H3: 24px (Bold)
- H4: 20px (SemiBold)

**Body:**
- Large: 18px (Regular)
- Normal: 16px (Regular)
- Small: 14px (Regular)
- Tiny: 12px (Medium)

**Buttons:**
- Large: 18px (SemiBold)
- Normal: 16px (SemiBold)
- Small: 14px (SemiBold)

## 🎨 LOGO GUIDELINES

### Logo Concept: "Checkmark List"

**Description for Canva:**
Create a logo with:
1. A bold checkmark (✓) in Listly Blue (#3B82F6)
2. Three horizontal lines beside/below the checkmark in Listly Purple (#8B5CF6)
3. The word "Listly" in Inter Bold below the icon
4. Rounded corners on all elements (30px radius)
5. Clean, modern, minimal style

### Logo Variations to Create

**1. Primary Logo (Icon + Text)**
- Checkmark + lines + "Listly" text
- Use on: Website header, app splash screen
- Size: 2000x500px

**2. Icon Only (Square)**
- Just checkmark + lines, no text
- Use on: App icon, favicon, social media profile
- Size: 1024x1024px

**3. Horizontal Logo**
- Icon on left, text on right
- Use on: Email signatures, presentations
- Size: 1200x300px

### Logo Colors

**Full Color:**
- Checkmark: Listly Blue (#3B82F6)
- Lines: Listly Purple (#8B5CF6)
- Text: Dark Gray (#111827)

**Monochrome (Dark):**
- All elements: Dark Gray (#111827)
- Use on: Light backgrounds

**Monochrome (Light):**
- All elements: White (#FFFFFF)
- Use on: Dark backgrounds

### Logo Don'ts

❌ Don't stretch or distort the logo
❌ Don't change the colors (except approved variations)
❌ Don't add drop shadows or effects
❌ Don't rotate the logo
❌ Don't place on busy backgrounds
❌ Don't use low-resolution versions

## 🎯 DESIGN STYLE GUIDE

### Visual Style

**Modern & Clean:**
- Lots of white space
- Minimal clutter
- Clear hierarchy
- Rounded corners everywhere

**Friendly & Approachable:**
- Soft colors (not harsh)
- Rounded shapes (not sharp)
- Conversational tone
- Helpful, not corporate

**Professional:**
- Consistent spacing
- Aligned elements
- Proper contrast
- High-quality images

### Photography Style

**If using photos:**
- Bright and airy
- Natural lighting
- Real people (not stock-looking)
- Grocery/food/shopping context
- Diverse representation

**Filters:**
- Slight brightness increase (+10%)
- Slight saturation increase (+5%)
- No heavy filters
- Keep it natural

### Icons Style

**Icon Set:**
- Lucide Icons (lucide.dev) - preferred
- Rounded style
- 2px stroke weight
- Consistent size (24px standard)
- Use Listly Blue or Purple

## 💡 CANVA PRO TIPS

### Using Brand Kit

1. **Apply Brand Colors:**
   - Click any element
   - Click color picker
   - Select from "Brand colors" section
   - Colors auto-populate from your brand kit

2. **Apply Brand Fonts:**
   - Select text
   - Font dropdown shows "Brand fonts" at top
   - One-click to apply

3. **Use Logo:**
   - Click "Uploads" or "Brand kit"
   - Drag logo onto design
   - Resize while holding Shift (maintains ratio)

4. **Create from Template:**
   - Save your designs as templates
   - Reuse for consistency
   - Update once, applies everywhere

### Keyboard Shortcuts

- **Duplicate:** Ctrl+D (Cmd+D on Mac)
- **Group:** Ctrl+G
- **Align:** Use alignment tools (top toolbar)
- **Copy style:** Alt+Click element, then click target
- **Resize proportionally:** Hold Shift while dragging

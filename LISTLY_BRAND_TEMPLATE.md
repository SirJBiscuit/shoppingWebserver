## Typography

### Primary Font: Inter

**Headings**
- Font: Inter Bold (700)
- Sizes: 48px (H1), 36px (H2), 24px (H3), 20px (H4)
- Letter spacing: -2%
- Line height: 1.2

**Body Text**
- Font: Inter Regular (400)
- Size: 16px (standard), 14px (small)
- Line height: 1.5
- Letter spacing: 0%

**Buttons**
- Font: Inter SemiBold (600)
- Size: 16px (standard), 14px (small)
- Letter spacing: 0%
- Text transform: None (sentence case)

### Font Alternatives
If Inter is unavailable:
1. Poppins
2. Montserrat
3. Open Sans
4. System default (San Francisco, Segoe UI, Roboto)

## Brand Voice & Tone

### Personality
- **Friendly** - Like a helpful friend
- **Clear** - No jargon, simple language
- **Encouraging** - Positive and supportive
- **Smart** - Intelligent but not condescending

### Writing Style
- Use contractions (you're, we'll, it's)
- Keep sentences short (under 20 words)
- Use active voice
- Be conversational
- Avoid technical terms

### Example Phrases

**Good:**
- "You're all set! 🎉"
- "Let's add some items"
- "Save $20 this week"
- "Your shopping just got easier"

**Bad:**
- "Configuration complete."
- "Please proceed to add items"
- "Potential savings: $20.00 USD"
- "Optimize your shopping experience"

## Logo Usage Guidelines

### Clear Space
- Minimum clear space: 50% of logo height on all sides
- Keep logo away from edges and other elements

### Minimum Size
- Digital: 32px height minimum
- Print: 0.5 inch height minimum

### Do's
✅ Use official logo files only  
✅ Maintain aspect ratio  
✅ Use on solid backgrounds  
✅ Ensure good contrast  
✅ Use approved color variations

### Don'ts
❌ Don't stretch or distort  
❌ Don't change colors (except approved variations)  
❌ Don't add effects (shadows, glows, etc.)  
❌ Don't rotate  
❌ Don't place on busy backgrounds  
❌ Don't recreate or modify
